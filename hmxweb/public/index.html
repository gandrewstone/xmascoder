What is this?

I have set up a Christmas tree with 800 individually controllable any color Christmas lights.

This web site lets you write programs to turn on the lights in cool patterns!

There's a "simulator" so you can see what your program might do.

Then you can click "Run on my tree!" to send it to the real thing (live stream is delayed for 15-30 seconds).

You can do it using a drag-n-drop language by Google called Blockly (similar to Scratch), or in Javascript.
